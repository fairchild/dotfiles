# Image Generation Assets

Use this directory for stable, reusable assets that help the skill generate or compare images.

Good candidates:

- reference images that are safe to keep in the repo
- prompt packs or evaluation prompt templates
- gallery HTML/CSS templates
- masks, layout guides, color palettes, or style boards

Do not store routine generated outputs here. Generated images belong in `../outputs/`, and local usage history belongs in `../data/`.
